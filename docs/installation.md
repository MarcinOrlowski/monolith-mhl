# Installation

Aside from recommended way of using [KDE Store](https://store.kde.org/p/2353773/),
here are alternative installation approaches:

## Manual installation

Alternatively, if you feel technical enough, you can grab a ZIP archive from the project
[Releases](https://github.com/MarcinOrlowski/monolith-mhl/releases) page, unpack it to your
`~/.local/share/plasma/wallpapers/` folder. Next open wallpaper settings window (if it was already
opened you must close it and re-open) and select `Monolith MHL`.

NOTE: If wallpaper is still not present in settings, you might need to restart your Plasma Shell:

```bash
systemctl --user restart plasma-plasmashell.service
```
