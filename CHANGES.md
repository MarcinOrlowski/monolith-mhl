# Changelog

## v1.2.0 (TBD)

- New post-filter "Mask", with per-region opacity controls for blending the
  mask color over the cut and gap regions.
- Added extensive documentation for built-in filters.
- Filter settings now include "Help" button linking to the docs.

## v1.1.0 (2026-04-17)

- Added Lava Lamp effect with 8 bundled color themes.
- Fixed Blur filter minimum distance changed to 2 pixels.
- Animation can be paused using desktop context menu.
- Added more documentation to the project.

## v1.0.0 (2026-03-27)

- Initial release with Rainbow Waves effect.
